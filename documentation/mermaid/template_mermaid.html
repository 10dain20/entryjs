mermaid 표기 룰

순서는 다음과 같습니다.

1. Entry 와의 상관관계를 작성,

표기법
Entry01 <--> TestClass01

2. import를 명시

표기법
Entry02 <|-- Class02 : import

3. 클래스 내의 관계도 명시

표기법
Class03 --> VARIABLES

4-1 생성자의 인자를 명시

표기법
VARIABLES : exampleVARIABLE

4-2 변수 명시

VARIABLES : exampleVarFromConstructor = constructor.parameter
## 만약 생성자와의 연관성이 있을 경우 위와같이 명시

일반적인경우에는
VARIABLES : exampleVARIABLE
위와같은 생성자 인자 명시와 같은방식으로

4-3 함수 명시

FUNCTIONS : test()
localFUNCTIONS : _testlocal()
## 만약 parameter나 포함되는 함수등 에 대해 그룹이 생성이 가능하며, 충분히 크다면 분리하여 다음과 같이 표기

예제
